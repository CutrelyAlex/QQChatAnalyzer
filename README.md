# 概述
本程序用于分析使用个人QQ导出的QQ群聊聊天记录，支持个人分析、群聊分析、社交网络可视化和 AI 驱动的摘要生成。
本程序不支持图片分析。

## 使用方法
1. 克隆或下载本项目到本地目录中
2. 确保Python环境存在，并安装依赖项：
```bash
python -m pip install -r requirements.txt
```
3. 配置项目，复制一份根目录下的`.env.example`重命名为`.env`，并按照注释修改其中的内容
4. 运行应用程序：
```bash
python app.py
```
5. 在浏览器中访问对应的本地地址，然后根据提示和需求进行使用即可。

## 聊天记录导出方法
1. 进入群聊页面，点击右下角的显示消息记录按钮
2. 点击展开页面右下角的"消息管理器"
3. 选择你想导出的群聊记录，右键并点击导出消息记录，选择格式为文本文件（txt），并保存到目录的`/texts/`目录下
4. 如果聊天记录较大，可能需要较长的时间进行分析（例如10MB大小的聊天记录的群聊分析可能需要40s）

# 合规提示
本程序接受通过合规途径，使用QQ的消息管理器导出的txt格式的文本聊天记录，不支持解析bak加密文件聊天记录。
请在导出聊天记录并进行分析前，确保你已经征得群员同意，并不使用本程序用于任何非法用途。
AI服务在不同模型下质量不同，且可能会出错，请在使用AI总功能结时确保你知晓风险，并征得群员同意。
